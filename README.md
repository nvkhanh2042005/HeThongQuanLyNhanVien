## Hệ Thống Quản Lý Nhân Viên – C++

## Giới thiệu

Đây là project xây dựng hệ thống quản lý nhân viên bằng ngôn ngữ C++ theo hướng lập trình hướng đối tượng (OOP).
Chương trình hỗ trợ quản lý thông tin nhân viên với các chức năng thêm, sửa, xóa, tìm kiếm, sắp xếp và lưu dữ liệu vào file.

Project được thực hiện nhằm rèn luyện kỹ năng lập trình C++, tư duy hướng đối tượng và xử lý dữ liệu thực tế.

## Chức năng chính

* Thêm nhân viên mới
* Hiển thị danh sách nhân viên
* Tìm kiếm nhân viên theo mã hoặc tên
* Cập nhật thông tin nhân viên
* Xóa nhân viên
* Sắp xếp nhân viên theo lương hoặc tên
* Đọc/Ghi dữ liệu từ file CSV
* Áp dụng kế thừa cho nhiều loại nhân viên

## Kiến thức và công nghệ sử dụng

* Ngôn ngữ C++
* Lập trình hướng đối tượng (OOP)
* Kế thừa và đa hình
* Xử lý file (CSV/TXT)
* STL Vector
* Thuật toán tìm kiếm và sắp xếp

## Cấu trúc project

* `main.cpp` → Chương trình chính
* `Employee.h / Employee.cpp` → Lớp nhân viên cơ bản
* `OfficeEmployee.h` → Nhân viên văn phòng
* `ProductionEmployee.h` → Nhân viên sản xuất
* `Company.h` → Quản lý danh sách nhân viên
* `employees.csv` → File lưu dữ liệu nhân viên

## Mục tiêu project

* Thực hành xây dựng ứng dụng quản lý bằng C++
* Hiểu rõ cách tổ chức chương trình theo OOP
* Rèn luyện xử lý dữ liệu và thao tác với file
* Làm project cá nhân phục vụ học tập và đưa vào CV/GitHub
